# PLC Program Translation for Verification Purposes


摘要  
可编程逻辑控制器通常采用IEC 61131标准定义的五种语言之一进行编程。虽然为每个程序单元选择合适的语言能为开发者带来优势，但这给验证方法带来了严峻挑战。本文通过分析和比较这些语言，证明结构化文本（ST）编程语言能高效且便捷地表征所有PLC语言，以满足形式化验证需求。此外，我们提出了一种从指令表（IL）到ST编程语言的翻译方法（针对西门子实现方案），并提供了正确性证明框架。这使得基于ST的PLCverif模型检测方法可应用于安全关键型PLC程序的验证。
