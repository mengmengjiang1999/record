# week21-20240715-20240721

如果一个大仓库pull下来非常困难，那么：

git clone http://github.com/large-repository --depth 1


打印rust的LLVM version：

rustc --version --verbose

碰到一个提示说需要git默认分支切换到master，虽然不知道为什么但是只能先这样了

另外还有一个提示是说在clone qemu的时候
Failed to connect to github.com port 443:connection timed out
解决方法是
git config --global --unset https.proxy
git config --global --unset http.proxy


后面提示说，需要llvm-config的版本大于rustc的版本，然后Makefile里面使用的llvm版本是15。安装18之后改成18即可。
