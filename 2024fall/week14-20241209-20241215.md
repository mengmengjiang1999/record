出现以下报错如何处理？

Dockerfile:1
--------------------
   1 | >>> FROM ubuntu:22.04 as fuzzware-base
   2 |     ENV LANG=C.UTF-8 LC_ALL=C.UTF-8
   3 |     RUN apt-get update && \
--------------------
ERROR: failed to solve: DeadlineExceeded: DeadlineExceeded: DeadlineExceeded: ubuntu:22.04: failed to resolve source metadata for docker.io/library/ubuntu:22.04: failed to do request: Head "https://registry-1.docker.io/v2/library/ubuntu/manifests/22.04": dial tcp 104.244.46.185:443: i/o timeout


经查找，发现原因是需要替换docker的镜像源。这个时候做下面的工作：

1.
sudo vim /etc/docker/daemon.json

添加以下内容：

{
  "registry-mirrors": [
    "https://hub-mirror.c.163.com",
    "https://ustc-edu-cn.mirror.aliyuncs.com",
    "https://ghcr.io",
    "https://mirror.baidubce.com"
  ]
}

2. 运行以下命令：

sudo systemctl daemon-reload
sudo systemctl restart docker

3. 运行

sudo docker info

检查是否完成信息更新


4. 重新运行Dockerfile
